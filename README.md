# Introductory Programming Exercises

This repository contains basic programming exercises designed to build problem-solving skills and familiarity with programming syntax.

## Key Focus Areas
- Simple program structure
- Basic input and output
- Fundamental logic implementation
- Practice-oriented exercises

## Overview
Created to strengthen foundational programming skills through hands-on exercises.


### Simple Calculator

A simple calculator implemented in Python that performs basic arithmetic operations and maintains a history of calculations using the Decimal module for precise arithmetic.

### Features

- **Basic Operations**: Addition, subtraction, multiplication, and division.
- **History Tracking**: Keeps a record of all calculations performed.
- **Error Handling**: Custom exceptions for handling errors like division by zero.
